DI in AngularJS

  • $provide allows us to register recipes for object creation
  • $injector gives access to singletons produced based on those recipes
  • Singleton instances are called "services"

2-phase lifecycle

  • Configure
  • Run

Modules accept various recipes

  • Values
  • Services
  • Factories
  • Constants
  • Providers
